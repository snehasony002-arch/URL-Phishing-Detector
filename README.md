 URL Phishing Detector Using Machine Learning

 Project Overview

This project is a Machine Learning-based URL Phishing Detection system developed using Python and Flask. The application analyzes website URLs and predicts whether they are legitimate or phishing based on URL characteristics.

The system also provides:

* Risk Score Analysis
* Low / Medium / High Risk Levels
* Recent Scan History
* Interactive Cybersecurity Interface
* Real-Time Prediction

 Technologies Used

* Python
* Flask
* HTML
* CSS
* Pandas
* Scikit-learn
* Joblib

Features

* Detects suspicious URLs
* Displays risk score indicators
* Shows Low, Medium and High risk levels
* Maintains recent scan history
* User-friendly interface
